# GPA Calculator

This is a simple command-line GPA Calculator application written in Java. It is intended to be a basic demonstration of object-oriented principles. The project structure is set up for future expansion into a more complete student management system.

## Folder Structure

- `src`: Contains the Java source files.
  - `GPAcalculator.java`: The main class that will drive the application.
  - `personMethods.java`: A class intended to hold methods related to a person.
  - `studentMethods.java`: A class for methods specific to students.
  - `teacherMethods.java`: A class for methods specific to teachers.
- `lib`: This folder is for dependencies, though this project has none.

The compiled `.class` files will be generated in the `bin` folder.

## How to Run

1.  Make sure you have a JDK installed.
2.  Open the `GPAcalculator` folder in VS Code.
3.  Go to `src/GPAcalculator.java`.
4.  Click the "Run" button above the `main` method or press `F5`.
5.  The program will start in the terminal. Currently, it displays a basic menu.
