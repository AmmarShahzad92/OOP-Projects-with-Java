# OOP Projects with Java

This repository is a collection of practical Java projects designed to demonstrate the real-world application of Object-Oriented Programming (OOP) concepts. Through distinct, small-scale software applications, this repository showcases key design principles like inheritance, polymorphism, encapsulation, and abstraction in action, making it a great learning resource.

## Projects

Here is a list of the projects included in this repository:

-   **[Accounts DBMS](AccountsDBMS/)**: A simple command-line Database Management System for bank accounts.
-   **[Array Swap](arraySwap/)**: A program that swaps the largest and smallest elements in an array.
-   **[GPA Calculator](GPAcalculator/)**: A basic structure for a GPA calculation application.
-   **[Grocery Store MS](GroceryStoreMS/)**: A command-line management system for a grocery store.
-   **[Intelligent Calculator](IntelligentCalculator/)**: A calculator that keeps a history of operations.
-   **[Matrix Calculator](MatrixCalculator/)**: A tool for performing matrix operations like addition and multiplication.
-   **[Number Array Editor](NumberArrayEditor/)**: An application for performing CRUD operations on an array of numbers.
-   **[Polynomial Adder](PolynomialAdder/)**: A program to add two polynomials.
-   **[Simple Calculator](SimpleCalculator/)**: A basic command-line calculator.
-   **[Student DBMS](StudentDBMS/)**: A command-line Database Management System for student records.

Each project folder contains its own `README.md` file with specific details about the project and instructions on how to run it.
