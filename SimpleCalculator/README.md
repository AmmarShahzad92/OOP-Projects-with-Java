# Simple Calculator

This is a basic command-line calculator application written in Java. It allows users to perform simple arithmetic operations: addition, subtraction, multiplication, and division. The program presents a menu to the user to choose an operation and then prompts for two numbers to perform the calculation on.

## Folder Structure

- `src`: Contains the Java source files.
  - `IntelligentCalculator.java`: The main class that handles user interaction and the menu.
  - `CalculatorFunctions.java`: A class that contains the methods for the arithmetic operations.
- `lib`: This folder is for dependencies, though this project has none.

The compiled `.class` files will be generated in the `bin` folder.

## How to Run

1.  Make sure you have a JDK installed.
2.  Open the `SimpleCalculator` folder in VS Code.
3.  Go to `src/IntelligentCalculator.java`.
4.  Click the "Run" button above the `main` method or press `F5`.
5.  The program will start in the terminal and display a menu of operations. Follow the prompts to perform calculations.
