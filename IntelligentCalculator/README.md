# Intelligent Calculator

This is a command-line calculator application written in Java that not only performs basic arithmetic operations but also maintains a history of the calculations. Users can view and manage this history.

## Features

-   **Arithmetic Operations:** Addition, subtraction, multiplication, and division.
-   **History Management:**
    -   View the entire calculation history.
    -   Delete all history records.
    -   Delete a specific history record by its position.

## Folder Structure

- `src`: Contains the Java source files.
  - `CalculatorInJava.java`: The main class that handles user interaction and the menu.
  - `CalcOperations.java`: A class that contains the methods for the arithmetic operations and history management.
- `lib`: This folder is for dependencies, though this project has none.

The compiled `.class` files will be generated in the `bin` folder.

## How to Run

1.  Make sure you have a JDK installed.
2.  Open the `IntelligentCalculator` folder in VS Code.
3.  Go to `src/CalculatorInJava.java`.
4.  Click the "Run" button above the `main` method or press `F5`.
5.  The program will start in the terminal and display a menu of operations. Follow the prompts to perform calculations and manage the history.
